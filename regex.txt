REGEX Quick Guide

^ - Begining of a line
$ - End of the line
. - Matches any character
\s - Matches whitespace
\S - Matches any non-whitespace character
* - Repeats a character 0 or more times
*? - Repeats characater 0 or more times (non-greedy)
+ - Repeats a character 1 or more times  (greater than or equal to 1)
+? - Repeats a character 1or more times (non-greedy)
[aeiou] - Matches a single character in the listed set
[^XYZ] - Matches a single character in the non-listed set
[a-z0-9] - The set of characters can include a range
( - Indicates where string extraction is to start
) - Indicates where string extractions is to end
